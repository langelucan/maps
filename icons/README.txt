INSTRUCTIONS POUR GÉNÉRER LES ICÔNES
=====================================

Les icônes doivent être créées pour l'extension Chrome.

Méthode 1: Utiliser le générateur HTML
---------------------------------------
1. Ouvrez le fichier "generate-icons.html" dans un navigateur
2. Cliquez sur chaque bouton "Télécharger" pour sauvegarder les icônes
3. Placez les fichiers icon16.png, icon32.png, icon48.png, icon128.png dans ce dossier

Méthode 2: Créer vos propres icônes
------------------------------------
Créez des fichiers PNG avec les dimensions suivantes:
- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

Recommandations:
- Utilisez un fond vert (#059669 ou #10b981)
- Ajoutez un symbole de marqueur de carte ou de localisation
- Gardez le design simple et reconnaissable
- Utilisez des couleurs contrastées (blanc sur vert fonctionne bien)

Méthode 3: Utiliser un outil en ligne
--------------------------------------
Vous pouvez utiliser des outils comme:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- Canva, Figma, ou Adobe Express

IMPORTANT: Sans les icônes, l'extension ne se chargera pas dans Chrome.
